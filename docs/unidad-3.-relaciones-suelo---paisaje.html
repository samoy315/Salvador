SUELOS DE EL SALVADOR

Unidad 3. Relaciones suelo - paisaje

En este tema estudiaremos los conceptos básicos relacionados con las relaciones suelo-paisaje, para entender el efecto de las geoformas como factor formador de suelos y su efecto en la diversidad y distribución de los suelos. Igualmente, revisaremos aspectos genéticos y taxonómicos de las geoformas, así como conceptos y los procesos fundamentales para la cartografía geomorfológica como herramienta para el mapeo de suelos. En el apartado de “Material Bibliografico” Se muestra una lista de documentos para los cuales se sugiere su lectura para complementar los conocimientos impartidos durante las clases teóricas. Bienvenidos al tema!!.
